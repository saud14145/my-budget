حدود الاسترداد الشهرية (الأهلي) — الشهر الحالي

مبني على الاسترداد الفعلي المرتبط من رسائل البنك، مصنّفًا حسب فئة كل محل. الفرنسي بدون حدود شهرية.

ملاحظات

تتحدث هذه اللوحة تلقائيًا فور تحليل رسائل جديدة من تبويب "لصق الرسائل". جميع البيانات محفوظة محليًا على متصفح هذا الجهاز فقط (localStorage) — استخدم زر "نسخة احتياطية" في تبويب الإعدادات بشكل دوري لتصدير نسخة JSON، خصوصًا قبل مسح بيانات المتصفح أو تغيير الجهاز.

لصق رسائل البنك

الصق أي عدد من الرسائل دفعة واحدة، رسالة تلو الأخرى كما وصلتك بالضبط. لا حاجة لفصلها بأسطر فارغة — النظام يكتشف بداية كل رسالة تلقائيًا.

النسخ الاحتياطي والبيانات

المزامنة التلقائية مع GitHub

تحفظ البيانات تلقائيًا كـ commit في مستودعك على GitHub — بدون تحميل أو رفع يدوي، وكل حفظة مسجّلة في سجل التغييرات (Commits) بالمستودع. يعمل هذا أيضًا كنسخة سحابية تظهر على أي جهاز تفتح منه نفس الصفحة.

لإنشاء Token: من إعدادات GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens → اختر هذا الريبو فقط → صلاحية Contents: Read and write. الـ Token يُحفظ فقط على هذا الجهاز ولا يُرسل لأي مكان غير GitHub.
